Feature: Product Search and Filter

User Story:
As a shopper, I want to search and filter products so that I can quickly find items that match my requirements.

Requirements:

1. Basic Search
   - Given I'm on the homepage
   - When I enter "laptop" in search box and press Enter
   - Then I should see all products containing "laptop" in name or description
   - And results should be sorted by relevance (default)
   - And I should see result count: "Showing X results for 'laptop'"

2. Search with No Results
   - Given I search for non-existent product
   - When search returns 0 results
   - Then I should see "No products found for 'xyz'"
   - And I should see suggested search terms
   - And I should see popular categories as alternatives

3. Price Range Filter
   - Given I'm viewing search results
   - When I apply price filter: Min=$500, Max=$1500
   - Then only products within price range should display
   - And price filter should show selected range
   - And result count should update

4. Multiple Filters (Category + Brand)
   - Given I'm viewing search results
   - When I select Category="Electronics" AND Brand="Samsung"
   - Then only products matching BOTH filters should display
   - And applied filters should show as removable chips
   - And I can remove individual filters by clicking X on chip

5. Sort Options
   - Given I'm viewing filtered results
   - When I select sort option
   - Then results should re-order:
     * Price: Low to High
     * Price: High to Low
     * Customer Rating
     * Newest First
     * Popularity

6. Filter Persistence
   - Given I've applied filters
   - When I click on a product and press back button
   - Then all my previous filters should still be applied
   - And scroll position should be maintained

7. Filter Combinations
   - Users can combine up to 5 filters simultaneously
   - Filter combinations should work with AND logic
   - Clear all filters button should reset to unfiltered state

8. Performance Requirements
   - Search results should load within 2 seconds
   - Filter application should be instant (<500ms)
   - Support minimum 10,000 products in catalog

9. Mobile Responsiveness
   - Filters should appear in slide-out panel on mobile
   - Search bar should be easily accessible at top
   - Applied filters should be visible without scrolling
